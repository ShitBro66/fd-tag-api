标签与图片接口文档汇总

来源：ShowDoc 页面接口；整理时间：2026-07-01。token 示例已省略。

接口总览

文档方法接口路径ShowDoc 标题职责
标签编辑POST/marketingTag/marketingTagUpdate标签增删改新增、编辑一级/二级标签。
标签详情GET/marketingTag/getTagById标签详情按 id 获取标签编辑回显。
标签树GET/marketingTag/tagTree标签树按国家代码获取一二级标签树。
标签关键字筛选GET/marketingTag/searchTags关键字筛选标签按关键字和国家代码筛选标签。
图片编辑POST/marketingTag/imageUpdate图片增删改，tagIds 传二级标签 id新增、编辑图片并绑定二级标签。
图片列表POST/marketingTag/imageList图片列表按二级标签查询图片列表，空参返回时间靠前的 20 张。
单图片详情GET/marketingTag/getImageById图片详情按 id 获取图片编辑回显。
标签排序POST/marketingTag/updateTagSort新接口拖拽后批量保存标签排序。

管理页使用点

管理页动作接口说明
打开页面、切换右上角国家GET /marketingTag/tagTree、POST /marketingTag/imageList标签树和图片列表都请求当前国家代码 countryCode；图片列表筛选时额外传 tagId。
新增/编辑标签POST /marketingTag/marketingTagUpdate新增/编辑提交 countryCode、parentId、tagName、isEnable。
编辑标签回显GET /marketingTag/getTagById请求带 id。
上传弹窗标签筛选GET /marketingTag/tagTree使用已加载标签树在前端按 tagName 本地过滤，不再调用 searchTags。
图片上传/编辑POST /marketingTag/imageUpdatetagIds 传二级标签 id 数组。
图片编辑回显GET /marketingTag/getImageById请求带 id。
标签拖拽排序POST /marketingTag/updateTagSort拖拽结束后提交数组，每项包含 id、sort。

1. 标签编辑

ShowDoc 标题：标签增删改 · 原文链接
POSThttps://products-api-dev.gs-souvenir.com/marketingTag/marketingTagUpdate

用于一级/二级标签新增、编辑。

JSON Body 字段

字段必填类型示例值备注
id编辑时是number2标签 id。
countryCode是stringus国家代码。
parentId是number0一级标签传 0；二级标签传一级标签 id。
tagName是stringChristmas Sale标签名。
isEnable是number1是否启用，二级标签不需要传值。

请求示例

{
  "id": 2,
  "countryCode": "us",
  "parentId": 1,
  "tagName": "Christmas Sale",
  "isEnable": null
}

返回示例

{
  "success": true,
  "code": 200,
  "data": true,
  "message": "success"
}

6. 图片列表

ShowDoc 标题：图片列表 · 说明：tagId 为空时展示时间靠前的二十张图片 · 原文链接
POSThttps://products-api-dev.gs-souvenir.com/marketingTag/imageList

JSON Body 字段

字段必填类型示例值备注

tagId是stringnull子标签 id；为空时返回时间靠前的 20 张图片。
current是number1当前页码。
pageSize是number9每页数量。

请求示例

{
  "countryCode": "us",
  "tagId": null,
  "current": 1,
  "pageSize": 9
}

  "code": 200,
  "data": {

      {
        "id": 2,
        "fullImageUrl": "https://cdn.example.com/full/demo.jpg",
        "thumbnailUrl": "https://cdn.example.com/thumb/demo.jpg",
        "tagIds": [2],
        "tagList": [
          {
            "id": 2,
            "parentId": 1,
            "tagName": "奖牌",
            "isEnable": null,
            "sort": 1,
            "imageId": 2,
            "children": []
          }
        ]
      }
    ],
    "total": 1,
    "size": 9,
    "current": 1,
    "pages": 1
  },
  "message": "success"
}
