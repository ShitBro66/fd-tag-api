管理页使用点分析

管理页位置/动作建议接口说明
打开标签管理页、刷新标签选择器GET /marketingTag/tagTree返回一二级标签树；一级标签包含启用状态，二级标签用于图片绑定。
新增/编辑标签POST /marketingTag/marketingTagUpdate一级标签 parentId=0；二级标签 parentId 传一级标签 id。ShowDoc 示例只列出新增字段，编辑/删除字段仍需后端确认。
查看一级标签编辑回显GET /marketingTag/getTagById按 id 获取一级标签详情，返回 tagName、isEnable、sort、language、tagNameTrans。
输入关键字查找标签GET /marketingTag/searchTags按 keyword 返回匹配标签列表，示例结果为二级标签。
新增/编辑图片并绑定标签POST /marketingTag/imageUpdatetagIds 传二级标签 id 数组；图片字段包含 fullImageUrl、thumbnailUrl。
图片编辑回显GET /marketingTag/getImageById返回图片 URL、tagIds 和 tagList，可直接回显已选标签。
图片列表/按标签筛选图片POST /marketingTag/imageListtagId 为空时展示时间靠前的 20 张图片；传二级标签 id 时按标签筛选。

仍需确认的问题

  1. marketingTagUpdate 标题为“增删改”，但 ShowDoc 请求体只给出新增字段；编辑是否传 id、删除是否传 isDel 仍需确认。
  2. imageUpdate 标题为“增删改”，但 ShowDoc 请求体只给出图片 URL 与 tagIds；编辑/删除字段仍需确认。
  3. searchTags 的 ShowDoc 示例把 token 同时放在 header 与 formdata；按现有项目约定建议只走统一 header token。

1. 标签新增/编辑/删除

ShowDoc 标题：标签增删改 · 原文链接
POSThttps://products-api-dev.gs-souvenir.com/marketingTag/marketingTagUpdate

Headers

字段必填类型示例值备注
token是string已省略：登录 token

Query 参数

无

JSON Body 字段

字段必填类型示例值备注
parentId是number0一级标签传 0；二级标签传一级标签 id。
tagName是string产品类型标签中文名。
isEnable是number1新增二级标签时不需要这个字段值。
sort是number1排序。
language是stringen-us语言。
tagNameTrans是stringProduct Type标签翻译名。

请求示例

{
  "parentId": 0,
  "tagName": "产品类型",
  "isEnable": 1,
  "sort": 1,
  "language": "en-us",
  "tagNameTrans": "Product Type"
}

返回示例

{
  "success": true,
  "code": 200,
  "data": true,
  "message": "success"
}
